chalhoub_2007

Chalhoub E, Hanson RW, Belovich JM.
A computer model of gluconeogenesis and lipid metabolism in the perfused liver.
Am J Physiol Endocrinol Metab. 2007 Dec;293(6):E1676-86. Epub 2007 Oct 2.
PubMed PMID: 17911349.
